# Windows 打包发布指南

## 📦 打包步骤

### 给打包操作者

如果你需要在 Windows 上打包此项目，请按以下步骤操作：

#### 方法一：使用自动化脚本（推荐）

1. 在 Windows 上克隆/复制项目到本地
2. 双击运行 `build.bat`
3. 等待打包完成（约 2-5 分钟）
4. 可执行文件在 `dist\微信小店客服助手\` 目录

#### 方法二：手动打包

```bash
# 1. 安装 Python 3.10+
# 2. 创建虚拟环境
python -m venv .venv
.venv\Scripts\activate

# 3. 安装依赖
pip install -r requirements.txt

# 4. 运行 PyInstaller
pyinstaller --clean weixin_store.spec
```

### 验证打包结果

打包完成后，检查以下内容：

1. **文件夹结构**：
   ```
   dist/微信小店客服助手/
   ├── 微信小店客服助手.exe
   ├── python311.dll (或类似版本)
   ├── PySide6/
   ├── config/
   └── images/
   ```

2. **测试运行**：
   - 双击 `微信小店客服助手.exe`
   - 确认程序能正常启动
   - 确认左侧面板按钮可用
   - 确认内置浏览器能加载页面

## 📤 分发给用户

### 方案一：完整文件夹分发（推荐）

将整个 `dist\微信小店客服助手\` 文件夹打包成 ZIP：

```bash
# 在 Windows 上
# 1. 右键 dist\微信小店客服助手 文件夹
# 2. 发送到 -> 压缩 (zipped) 文件夹
# 3. 生成 微信小店客服助手.zip
```

分发给用户后，用户只需：
1. 解压 ZIP 文件
2. 双击运行 `微信小店客服助手.exe`

### 方案二：安装程序（高级）

可以使用以下工具创建安装程序：

- **NSIS** - 免费，功能强大
- **Advanced Installer** - 商业，易用

## 🔧 常见问题排查

### 问题：缺少 VCRUNTIME140.dll

**解决方案**：
- 让用户安装 [Visual C++ Redistributable](https://aka.ms/vs/17/release/vc_redist.x64.exe)
- 或在打包时添加 `--onefile` 选项（但会增加启动时间）

### 问题：杀毒软件误报

**解决方案**：
- 将程序目录添加到杀毒软件白名单
- 使用代码签名证书（商业方案）

### 问题：程序启动后闪退

**解决方案**：
- 检查 `--windowed` 模式下看不到错误，可临时改为 `--console` 查看错误
- 检查用户数据目录是否有写入权限

## 📊 打包文件大小参考

- **完整文件夹**：约 150-200 MB
- **压缩后的 ZIP**：约 80-120 MB
- **单个 exe（--onefile）**：约 60-80 MB（但解压慢）
