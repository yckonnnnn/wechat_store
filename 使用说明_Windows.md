# 微信小店客服助手 - Windows 使用说明

## 📦 快速开始

### 方法一：直接使用打包版（推荐）

如果你收到的是已经打包好的文件夹 `微信小店客服助手`：

1. **解压/复制文件夹**到任意位置（建议：`C:\Program Files\微信小店客服助手` 或桌面）
2. **双击运行** `微信小店客服助手.exe`
3. 首次启动会自动创建配置文件到用户数据目录

### 方法二：源码打包

如果你收到的是源码包：

1. **双击运行** `build.bat` 脚本
2. 等待打包完成（约 2-5 分钟）
3. 在 `dist\微信小店客服助手\` 目录找到可执行文件

---

## 🔧 系统要求

- **操作系统**: Windows 10/11 (64 位)
- **内存**: 至少 4GB 可用内存
- **磁盘空间**: 至少 500MB 可用空间
- **网络**: 需要网络连接以调用 LLM API

## ⚙️ 配置说明

### 首次使用配置

1. **配置模型 API**
   - 启动程序后，点击左侧菜单「模型配置」
   - 选择你使用的模型服务商（ChatGPT/Gemini/DeepSeek/豆包/Kimi 等）
   - 填入 API Key
   - 点击「测试连接」验证

2. **配置知识库**
   - 点击左侧菜单「知识库管理」
   - 可以添加/编辑/导入常见问题和答案
   - 支持批量导入 Excel/JSON 文件

3. **配置图片/视频**
   - 点击左侧菜单「图片与视频管理」
   - 上传需要发送的图片/视频
   - 设置触发关键词

### 配置文件位置

用户数据存储在：
```
C:\Users\你的用户名\AppData\Roaming\Annel AI 客服\
```

包含：
- `config/model_settings.json` - 模型配置
- `config/knowledge_base.json` - 知识库
- `config/agent_memory.json` - 会话记忆
- `logs/` - 日志文件

## 🚀 使用流程

1. **启动程序** - 双击 `微信小店客服助手.exe`

2. **登录微信小店**
   - 在内置浏览器中访问微信小店客服后台
   - 扫码登录

3. **启动自动服务**
   - 点击左侧「启动服务」按钮
   - 系统会自动：
     - 每 4 秒扫描未读消息
     - 自动点击进入会话
     - 抓取聊天记录
     - Agent 智能决策回复
     - 发送文本/图片/视频

4. **监控状态**
   - 左侧面板显示运行日志
   - 可查看 Agent 决策过程

## 🛠️ 常见问题

### Q: 程序无法启动？
A: 检查是否被杀毒软件拦截，尝试将程序目录添加到白名单

### Q: 无法连接微信？
A: 确保网络连接正常，微信小店页面可能需要手动刷新

### Q: API 调用失败？
A: 检查 API Key 是否正确，网络是否可以访问模型服务商

### Q: 回复不准确？
A:
- 检查知识库是否包含相关问题
- 调整系统提示词（docs 目录下）
- 在「模型配置」中调整模型参数

### Q: 如何备份配置？
A: 备份 `C:\Users\你的用户名\AppData\Roaming\Annel AI 客服\config\` 目录
